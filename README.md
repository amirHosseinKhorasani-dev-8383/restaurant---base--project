# Restaurant Website (Inspired UI Project)

This project is a personal front-end project inspired by a real restaurant website design.
The entire codebase was written by me from scratch without following a course or instructor.

## Technologies
- HTML
- Tailwind CSS
- JavaScript

## Description
A modern single-page restaurant website built as a personal practice project.
The focus of this project is on clean UI design, responsive layout, and basic interactivity using JavaScript.

## Features
- Fully responsive design with Tailwind CSS
- Clean and modern restaurant UI
- JavaScript-based interactions
- Single-page layout
